Custom Noise

Technical Details

▼

This generator creates white noise by generating random audio samples. This sound contains all audible frequencies at equal intensity.

The High Frequencies (Hiss) slider is a Low-Pass Filter. It blocks frequencies above the selected value. To get brown noise, this filter is set to a low value (300 Hz).

The Low Frequencies (Rumble) slider is a High-Pass Filter. It blocks frequencies below the selected value. To get violet noise, this filter is set to a high value (4000 Hz).

Adjusting both filters lets you create a wide spectrum of "colored" noise, from a deep rumble to a sharp hiss, or a perfect blend of both.
